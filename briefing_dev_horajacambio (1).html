01 — Funcionalidades

O que desenvolver

Conversor de Moedas
ALTA
  • Página em /conversor
  • Input numérico com máscara monetária — calcular ao digitar (sem submit)
  • Dropdown de moedas com bandeiras: USD, EUR, GBP, ARS, CAD, CHF, JPY, CNY
  • Botão inverter origem/destino com animação
  • Toggle para incluir IOF: Cartão 6,38% · Espécie 0,38% · Transferência 0,38%
  • Mini tabela com últimas 5 cotações do dia
  • Biblioteca de máscara sugerida: IMask.js ou Cleave.js
GET /last/USD-BRL ← cotação em tempo real
Calculadora de Viagem
ALTA
  • Campos: país de destino · valor em moeda local · tipo de pagamento
  • Calcular: valor base + IOF + spread = total em BRL
  • Botão "Compartilhar" via Web Share API nativa (funciona no mobile)
Gráfico de Variação Histórica
ALTA
  • Gráfico de linha nas páginas de cada moeda
  • Filtros: 7 · 30 · 90 · 365 dias
  • Tooltip com data, mínima e máxima ao passar o mouse
  • Biblioteca sugerida: Chart.js (ou Recharts se for React)
GET /json/daily/USD-BRL/{dias} ← trocar {dias} pelo filtro
Alertas de Cotação
MÉDIA
  • Formulário: moeda + valor alvo + direção (acima / abaixo)
  • Implementar com Web Push API + Service Worker
  • Verificar cotação a cada 10 min (mesmo intervalo atual)
  • Salvar alertas no localStorage
  • Limitar a 3 alertas simultâneos por usuário
  • ⚠️ Verificar se já existe service worker registrado antes de implementar
Painel de Mercados Globais
MÉDIA
  • Widget na home: Ibovespa, S&P 500, Nasdaq, Ouro, Bitcoin
  • Indicar visualmente se mercado está aberto (verde) ou fechado (cinza)
  • Countdown para próxima abertura/fechamento
  • API sugerida: Alpha Vantage (gratuita com limite)
Relógio de Mercados Mundiais
MÉDIA
  • Horário em tempo real de: São Paulo · Nova York · Londres · Frankfurt · Tóquio · Xangai
  • Usar Intl.DateTimeFormat nativo — sem biblioteca externa
  • Indicar se o mercado local está aberto ou fechado

02 — SEO

SEO e Performance

  • Meta tags dinâmicas com cotação atual no título: "Dólar hoje: R$ X,XX — HoraJá"
  • Novas páginas de conteúdo: /o-que-e-iof · /como-enviar-dinheiro-exterior · /melhor-hora-comprar-dolar
  • Score Lighthouse > 90 no mobile e desktop
  • Lazy loading em gráficos e seções abaixo do fold
  • Manter cache de API em no máximo 10 minutos
03 — Anúncios

Espaços para Google AdSense

  • Banner 728×90 — abaixo do menu (desktop)
  • Banner 300×250 — sidebar direita (desktop)
  • Banner 320×50 — entre seções (mobile)
  • Deixar os espaços reservados no layout aguardando aprovação do AdSense
